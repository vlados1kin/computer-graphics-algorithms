## Алгоритмы компьютерной графики
1. Разработать парсер файлов формата .obj</br>
2. Реализовать матричные преобразования координат из пространства модели в мировое пространство</br>
3. Реализовать матричное преобразование координат из мирового пространства в пространство наблюдателя</br>
4. Реализовать матричное преобразование координат из пространства наблюдателя в пространство проекции</br>
5. Реализовать матричное преобразование координат из пространства проекции в пространство окна просмотра</br>
6. Реализовать отрисовку проволочной 3D модели
